# Popuri Jayesh Portfolio

## 🚀 Welcome to Your Premium Developer Portfolio

We built an **Awwwards-Level**, highly-animated, and ultra-smooth modern portfolio just for you! 

### Features Implemented
- **Ghibli Style Loading Screen:** Includes a bouncing Ghibli avatar over a custom loading progress bar with spark particles.
- **Glassmorphism & Neon UI:** Soft `#3cff7a` accent glows across contact borders, customized cards, and background blur vectors.
- **Framer Motion Mastery:** Buttery smooth animations everywhere—scroll reveals, hero fade-ups, staggered service card lifts, and 3D tilting project frames.
- **Custom Cursor Interaction:** A smart cursor that expands when hovering over buttons and shows a 'VIEW' icon across project cards.
- **Contact & WhatsApp Magic:** Upon sending a message, a confetti explosion combined with a floating avatar popup occurs—meanwhile sending background emails via FormSubmit and redirecting users instantly to WhatsApp.

### Tech Stack
Built utilizing raw CSS, TailwindCSS v4, React 19, Next.js 15, Framer Motion, and GSAP principles.

### Run Local
```bash
npm run dev
```
